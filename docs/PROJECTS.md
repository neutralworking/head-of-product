- **Name** — display name
- **Repo** — GitHub repo or other canonical code/content home
- **Tier** — `now` / `next` / `later` / `parked` / `archived`
- **Done** — definition of done for the current effort
- **Monetisation** — thesis. If none, say so.

- **Revenue status** — `active` (earning) / `path` (plausible route documented) / `none` (no path)

Legend for empty cells: `—` means none / unknown / not applicable. `?` means HoP needs to confirm.
